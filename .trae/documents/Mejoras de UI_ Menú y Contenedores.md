Voy a realizar mejoras visuales en el menú lateral y en la presentación de las tarjetas (contenedores) para que la página luzca más moderna, organizada y atractiva, siguiendo tus indicaciones.

### 1. Mejoras en el Menú Lateral (`css/main.css`)
El objetivo es hacerlo más legible y estético, con una clara distinción de la opción activa.
- **Títulos de Sección (`h3`):** Aumentaré ligeramente el tamaño y el contraste para que se lean mejor, dándoles un toque de color (probablemente el tono primario o acento).
- **Ítems del Menú:**
  - Mejorar el estado **Hover**: Un fondo más suave y evidente.
  - Mejorar el estado **Activo**: Reemplazar el punto a la derecha por un estilo más moderno (ej. un borde lateral izquierdo de color neón o un fondo degradado sutil) para que quede claro dónde está el usuario.
  - Ajustar el espaciado para que no se sienta ni muy apretado ni muy disperso.

### 2. Mejoras en los Contenedores/Tarjetas (`css/main.css`)
El objetivo es equilibrar el espacio y diferenciar mejor las tarjetas entre sí.
- **Espaciado (Grid):** Ajustaré el `gap` (espacio entre tarjetas) en `.theory-nav-grid` para equilibrarlo (actualmente 2rem, probaré con un valor intermedio o responsivo para que se vea unido pero no amontonado).
- **Estilo de las Tarjetas (`.theory-card`):**
  - **Diferenciación:** Añadiré un borde superior de color (o un degradado sutil) para que cada tarjeta destaque individualmente.
  - **Profundidad:** Mejorar el efecto de `box-shadow` y el `glassmorphism` (efecto cristal) para que resalten sobre el fondo oscuro.
  - **Botones "Ver más":** Asegurar que tengan una posición y estilo consistentes.

### 3. Detalles Visuales Generales
- **Tipografía:** Asegurar que los textos dentro de las tarjetas tengan buena jerarquía (título vs descripción).
- **Bordes:** Suavizar los bordes (`border-radius`) para mantener la coherencia con el diseño "Glassmorphism".
